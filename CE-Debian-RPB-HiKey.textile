h2. Install Instructions

For the bootloader files: "https://github.com/96boards/documentation/wiki/Reference-Bootloader-Hikey#installing-from-the-pre-built-binaries":https://github.com/96boards/documentation/wiki/Reference-Bootloader-Hikey#installing-from-the-pre-built-binaries

Look for the desirable release at "https://builds.96boards.org/snapshots/reference-platform/debian/":https://builds.96boards.org/snapshots/reference-platform/debian/

h3. Downloading

With the version in hands, simply download the images with wget:

bc. wget https://builds.96boards.org/snapshots/reference-platform/debian/30/hikey/hikey-boot-linux-20151106-30.uefi.img.gz
wget https://builds.96boards.org/snapshots/reference-platform/debian/30/hikey/hikey-rootfs-debian-jessie-alip-20151106-30.emmc.img.gz
gunzip hikey-*

h3. Flashing

Boot into fastboot mode, then just flash the boot and rootfs files:

bc. sudo fastboot flash boot hikey-boot-linux-*.uefi.img
sudo fastboot flash system hikey-rootfs-debian-jessie-alip-*.emmc.img

Now just reboot the board and enjoy the release :-)

For known issues and more information, please check "https://github.com/96boards/documentation/wiki/ReferenceSoftware":https://github.com/96boards/documentation/wiki/ReferenceSoftware
